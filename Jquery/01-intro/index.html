Practical 01 - What is jQuery & Basic Usage

1. What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library.

It makes things like HTML document traversal, manipulation, event handling, and animation much simpler.


jQuery Motto: "Write less, do more"

2. jQuery Syntax

Basic Syntax: $(selector).action()

  • $ - jQuery selector
  • selector - HTML element to select
  • action() - jQuery method to perform

3. Document Ready Event

Always wait for document to load before using jQuery:

$(document).ready(function(){ ... });

Or shorthand: $(function(){ ... });

5. jQuery vs JavaScript

Hide element in JavaScript:

document.getElementById("test").style.display = "none";

Hide element in jQuery:

$("#test").hide();

jQuery is much simpler and cleaner!

6. jQuery Key Features

  • Easy to learn and use
  • Cross-browser compatibility
  • DOM manipulation made easy
  • Event handling simplified
  • AJAX support
  • Animation and effects
  • Large plugin ecosystem
