# HTML Task 05 - Tables 

Learn HTML table structure and features with accessibility best practices.

## Objective

Master HTML table structure and features:
- `<table>`, `<thead>`, `<tbody>`, `<tfoot>` elements
- `<caption>` for table titles
- `colspan` and `rowspan` for cell merging
- Accessibility attributes: `scope`

## Topics 

1. **Basic Table Structure** - `<table>`, `<tr>`, `<th>`, `<td>`
2. **Table Caption** - Adding titles with `<caption>`
3. **Colspan & Rowspan** - Merging cells horizontally and vertically
4. **Table Sections** - `<thead>`, `<tbody>`, `<tfoot>`
5. **Accessibility** - `scope` attribute for screen readers
6. **Bootstrap Styling** - Striped, bordered, hover effects
7. **Responsive Tables** - Mobile-friendly implementations

## Accessibility Best Practices

Use `<caption>` to describe table content  
Add `scope` attribute to `<th>` elements  
Use semantic sections: `<thead>`, `<tbody>`, `<tfoot>`  
Use `<th>` for headers, not bold `<td>`  

**Scope Values:** `col`, `row`, `colgroup`, `rowgroup`
