## Form Elements

- `<form>` - Form container
- `<input>` - Input field
- `<label>` - Field label
- `<textarea>` - Multi-line text
- `<select>` - Dropdown
- `<option>` - Dropdown option
- `<button>` - Button
- `<fieldset>` - Group fields
- `<legend>` - Fieldset title
