HTML Attributes — Task 09

1. Global Attributes

These attributes can be used on almost all HTML elements:

  • id — unique identifier
  • class — used for styling and JS
  • title — tooltip text
  • style — inline CSS
  • hidden — hides the element
  • contenteditable — make element editable
  • draggable — enable drag behavior
  • tabindex — keyboard focus order
