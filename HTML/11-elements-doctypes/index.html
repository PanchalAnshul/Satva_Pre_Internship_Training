DOCTYPE and HTML Elements

1. DOCTYPE

The DOCTYPE for HTML5 is:

<!doctype html>

It tells the browser to use **standards mode** instead of quirks mode.

2. Block-level elements

Block elements start on a new line and take full width.

  • <div>
  • <p>
  • <h1>–<h6>
  • <section>
  • <article>

Example:

I am a block element (DIV)

3. Inline elements

Inline elements do not start on a new line.

  • <span>
  • <a>
  • <strong>
  • <em>

Example:

This is an inline SPAN inside a sentence.

4. Self-closing (void) elements

These elements have no closing tag:

  • <br>
  • <hr>
  • <img>
  • <meta>
  • <input>

Example:

Line 1
Line 2 (because <br> is used)

5. Semantic vs Non-semantic elements

Semantic elements describe meaning:

  • <header>
  • <nav>
  • <main>
  • <section>
  • <footer>

Non-semantic elements do NOT describe meaning:

  • <div>
  • <span>

Open semantic.html for examples.
