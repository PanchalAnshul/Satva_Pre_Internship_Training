# Task 11 — Elements & DOCTYPE

## What I learned
- HTML5 DOCTYPE: `<!doctype html>` puts browser in standards mode.
- Block-level elements take full width and start on a new line.
- Inline elements do NOT break the line.
- Self-closing elements have no closing tag.
- Semantic elements give meaning to structure (header, nav, section, article).
- Non-semantic elements (div, span) used for structure only.
