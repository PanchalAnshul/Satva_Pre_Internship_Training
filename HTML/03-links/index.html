Relative links (local files)

If you have another file in the same folder (for example other.html), link it with: <a href="other.html">Open other page</a>

Example linking a sibling file (you can create a small other.html for practice): Open other.html (relative)

mailto and tel links

Email link (opens email client): Email me

Telephone (mobile devices will prompt a call): Call: +91 1234567890

Download attribute

If you put a file `sample.pdf` in the same folder, this link makes the browser download it: Download sample.pdf

Tips: use rel="noopener noreferrer" with target="_blank" for security. Use absolute URLs for external resources and relative URLs for files inside your repo.
