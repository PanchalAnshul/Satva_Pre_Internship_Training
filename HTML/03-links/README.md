# Task 03 — Links

## Task 02 —HTML Links (CSS + Bootstrap)
- `<a>` tag is used to create hyperlinks.
- `href` defines the link destination.
- `target="_blank"` opens link in a new tab.
- `mailto:` opens the email client.
- Buttons can act as links using JavaScript.
- `title` attribute shows tooltip text on hover.
