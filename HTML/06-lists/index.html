HTML Lists

Ordered, Unordered & Description Lists

1. Unordered List (Bullets)

Tags: <ul> <li> | Use: List items without specific order
Code: 
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
Output:
  • HTML
  • CSS
  • JavaScript
List Style Types (CSS):

disc (default)

  • Item 1
  • Item 2

circle

  • Item 1
  • Item 2

square

  • Item 1
  • Item 2

2. Ordered List (Numbers)

Tags: <ol> <li> | Use: List items in specific order
Code: 
<ol>
  <li>Open VS Code</li>
  <li>Write HTML code</li>
  <li>Save file</li>
</ol>
Output:
  1. Open VS Code
  2. Write HTML code
  3. Save file
Type Attribute:

type="1" (default)

  1. First
  2. Second

type="A"

  1. First
  2. Second

type="a"

  1. First
  2. Second

type="I"

  1. First
  2. Second

type="i"

  1. First
  2. Second

start="5"

  1. Fifth
  2. Sixth

3. Description List (Terms & Definitions)

Tags: <dl> <dt> <dd> | Use: Define terms and descriptions
Code: 
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
Output:
HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JavaScript
Programming language for web interactivity

4. Nested Lists

Lists inside lists - Perfect for hierarchical data
Code: 
<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>Backend</li>
</ul>
Output:
  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python
  • Database
    • MongoDB
    • MySQL
