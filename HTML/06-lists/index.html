HTML Lists — Task 06

Unordered list (<ul>)

Use for lists where order doesn't matter.

  • Milk
  • Bread
  • Eggs

Ordered list (<ol>)

Use when sequence or priority matters.

  1. Install Node.js
  2. Set up project
  3. Run tests

Description list (<dl>)

Best for term → definition pairs.

HTML
HyperText Markup Language — structure of web pages.
CSS
Cascading Style Sheets — presentation and layout.
JS
JavaScript — interactive behavior.

Nested lists

Combine <ul> or <ol> inside list items.

  • Fruits
    1. Apple
    2. Banana
  • Vegetables
    • Carrot
    • Spinach

Inline / horizontal list

Useful for navs or tags.

  • Home
  • About
  • Contact

Accessiblity tips

  • Use semantic lists for screen readers.
  • Use descriptive text in list items — not just links like “click here”.
  • For long lists, provide headings and grouping.

Try editing list styles in DevTools (list-style-type, margin, padding) to see how layout changes.
