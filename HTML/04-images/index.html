HTML Images — Task 04

1. Basic image with alt

Always include alt text for accessibility.

2. Responsive image (srcset)

Browsers will pick the best image based on device pixel ratio or width.

Use srcset for @2x / high-DPI images.

3. Picture element for art direction

Serve different images for different viewport widths.

The browser uses the first matching <source>.

4. Decorative image (empty alt)

If an image is purely decorative, use an empty alt: alt="".

Screen readers will ignore images with empty alt.

5. Image lazy-loading (native)

Use loading="lazy" to defer offscreen images.

Improves page performance for large pages.

Open the page in a browser and inspect images with DevTools (Network tab) to see which file was loaded.
