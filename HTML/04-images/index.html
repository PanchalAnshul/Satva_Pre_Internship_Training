HTML Images

1. Using <img> Element

The <img> tag is used to display images in HTML.

2. src and alt Attributes

src defines the image URL and alt provides alternate text if the image cannot load.

3. Image Size using Width & Height

Image size can be controlled using HTML attributes or CSS properties.

4. Floating Image Left

The CSS float: left property allows text to wrap around the image on the right side. This is commonly used in articles and blogs.

5. Floating Image Right

The CSS float: right property allows text to wrap around the image on the left side. Floating images helps in better layout design.
